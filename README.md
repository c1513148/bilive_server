### 自行编译
  * 第一次使用
    1. 安装[Git](https://git-scm.com/downloads)
    2. 安装[Node.js](https://nodejs.org/)
    3. 命令行 `git clone https://github.com/Blokura/bilive_server.git`
    4. 命令行 `cd bilive_server`
    5. 命令行 `mkdir options`
    6. 命令行 `cp nedb/roomList.db options/roomList.db`
    7. 命令行 `npm install`
    8. 命令行 `npm run build`
    9. 命令行 `npm start`
  * 版本更新
    1. 定位到目录
    2. 命令行 `git pull`
    3. 命令行 `npm install`
    4. 命令行 `npm run build`
    5. 命令行 `npm start`
